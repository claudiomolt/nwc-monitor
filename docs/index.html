Features
Everything an agent needs
to operate on Lightning
From basic balance checks to automated payment reactions — NWC Monitor is the complete Lightning toolkit for autonomous agents.
⚡

Send & Receive

Pay invoices, pay Lightning addresses, create invoices to receive. Full bidirectional Lightning capability.

🎯

Action Triggers

Monitor incoming payments and fire actions automatically — send an email, WhatsApp message, Discord alert, log to a database, save to a file, or run any custom command.

🔑

Multi-Wallet Support

Manage multiple wallets simultaneously. Auto-detects Alby CLI wallets, accepts NWC strings, or creates disposable wallets instantly.

🧠

Smart Setup

Zero-config for Alby users. Paste an NWC string for any wallet. Or create one in one command with lncurl. Works for beginners and power users.

🛡️

Runs as a Service

Installs as a systemd service for reliable, persistent monitoring. Falls back to nohup if systemd isn't available.

🤖

Agent-Native

Designed for AI agents from day one. Tested with Anthropic Claude and OpenAI Codex. Works in sandboxed or host environments.
