Features

Full Lightning Operations

Balance checks, pay invoices, pay Lightning addresses, create invoices to receive, and list recent transactions — all via simple commands.

Wallet Setup

Three ways to connect.

NWC Monitor meets you where you are — beginner or power user.

🔍 Auto-detect Alby

Already using Alby CLI? NWC Monitor finds your wallets automatically. Zero configuration needed.

alby account ls 📋

🔑 Paste NWC String

Have a Nostr Wallet Connect string? Paste it directly. Compatible with any NWC-compatible wallet.

nostr+walletconnect://...

✨ Create one instantly

No wallet? One command creates a disposable Lightning wallet. No signup, no KYC, instant sats.

curl -s -X POST https://lncurl.lol 📋

Powered by lncurl.lol ⚡

🔧 Run as service

Deploy as systemd unit for production reliability. Or use nohup for quick background operation.

nwc-monitor --daemon 📋
